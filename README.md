**✨ ARTISAN — Premium Indulgence Platform ✨**

ARTISAN is an ultra-luxury, subscription-first food & wellness platform—crafted as a modern React frontend showcase where motion, interaction, and performance lead the experience.

This project is intentionally focused on high-fidelity UI engineering, using contemporary React patterns and advanced animation techniques to create immersive, tactile interfaces.

**What makes ARTISAN stand out**

Scroll-driven, immersive product journeys powered by Framer Motion

Smooth, inertia-based scrolling using React Lenis for a cinematic feel

A bespoke cursor interaction system designed to make navigation feel intentional

Dynamic theme transitions and motion-led storytelling

Animated, stateful product cards with curated filtering experiences

Tiered membership journeys with wellness-focused dashboards

**Tech Stack**

React 19 · TypeScript · Vite

Framer Motion v12 — advanced motion systems

React Router DOM v7 — fluid SPA navigation

Tailwind CSS with clsx & tailwind-merge for scalable styling

Lucide React Icons — minimal, modern iconography

**Intent**

This build reflects my focus on React frontend engineering, motion systems, and product-driven UI design—treating animation and interaction as core UX features, not visual garnish.
